Create ftp accounts and email accounts from your wordpress site.

== Description ==

Enables the Wordpress admin or user to create an email account and/or ftp account on their cpanel from their Wordpress site.

== Installation ==

1. Upload the `cpanel_ops` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Set its permission (roles).
4. Enter the cpanel information in the Cpanel settings page under settings section.
3. Go to widgets and drag and drop it to the sidebar widget

== Frequently Asked Questions ==
= What are the supported operations? =
= Currently supported operations: =
1. Create new ftp account.
2. Create new email account.
* The plugin provides a widget that when enabled provides access to the add ftp account page and the add email account page.
